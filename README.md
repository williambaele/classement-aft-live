## Classement AFT

This applicaiton helps the belgian tennis player to have a visual image of his season.

- Chart of victories/defeats
- Actual ranking
- Point value
- Prediction of the next ranking
- Chart with this past rankings
- All matches history


# Techs: NextJS + TailwindCSS
